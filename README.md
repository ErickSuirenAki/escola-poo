# Sistema de Gestão Acadêmica (POO)

## Descrição
Este projeto consiste em um sistema de gestão acadêmica desenvolvido em **Java**, utilizando os conceitos de **Programação Orientada a Objetos (POO)**.

O sistema simula o funcionamento de uma instituição de ensino, permitindo o gerenciamento de alunos, professores, disciplinas e notas.

O trabalho foi desenvolvido como atividade da disciplina de POO.

---

## Funcionalidades

O sistema permite:

- 👨‍🎓 Cadastro de alunos
- 🧑‍🏫 Cadastro de professores
- 📚 Criação de disciplinas
- 📝 Matrícula de alunos em disciplinas (máx. 6)
- 📊 Lançamento de notas (2 por disciplina)
- 📄 Visualização de histórico do aluno
- 📋 Visualização de notas por disciplina
- 🔎 Consulta de notas de um aluno específico

---

## 🛠️ Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Estruturas de dados:
  - ArrayList
  - HashMap
